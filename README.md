## Description
Simple global sound player for Godot,<br>
that lets you play sound anywhere without AudioStreamPlayer(2D/3D) nodes.
### Installation
```shell
cd addons
git submodule add https://github.com/ChloePrime/ChloePrime.Godot.SimpleGlobalSound
```
### Usage
See [example.tscn](example.tscn) and [example.gd](example.gd) for usage example.
